## 摘要
随着语言模型越来越多地部署在真实应用中，从模型中擦除特定知识的能力对于安全性和合规性至关重要。主流方法通过更新模型参数来寻求持久性擦除，但目标知识往往可以通过对抗性提示或重新学习被恢复。在本工作中，我们假设这一限制部分源于现有方法忽略了嵌入层。针对这一问题，我们提出了嵌入擦除模块 EMBER（EMBedding ERasure），该模块利用稀疏矩阵分解，从 token 嵌入中精确擦除概念相关特征。通过在 Gemma-2-2B-it 和 Llama-3.1-8B-Instruct 上对多种概念进行全面评估，我们发现，在现有方法基础上加入 EMBER，能够持续提升擦除效果和特异性，且一致性损失极小。此外，它极大增强了对重新学习的鲁棒性，将恢复的准确率降低多达 50%，在 Llama 上将其限制在 35%，而此前方法为 70%-76%。进一步分析表明，一致性代价是局部的，仅影响一小部分概念专属 token。我们的工作证实，精确的嵌入级干预对于鲁棒的概念擦除是必要的，并展示了现有方法可从此类增强中获益。

## Abstract
As language models are increasingly deployed in real-world applications, the ability to erase specific knowledge from them becomes critical for safety and compliance. Prominent methods seek persistent removal by updating the model's parameters, yet the target knowledge often can be recovered through adversarial prompting or relearning. In this work, we hypothesize this limitation stems in part from existing methods overlooking the embedding layer. To address this, we introduce EMBedding ERasure (EMBER), a plug-n-play erasure module that leverages Sparse Matrix Factorization for precise erasure of concept-related features from token embeddings. Through comprehensive evaluations across diverse concepts on Gemma-2-2B-it and Llama-3.1-8B-Instruct, we find that augmenting existing methods with EMBER consistently improves erasure efficacy and specificity across task formats, with minimal coherence loss. Moreover, it dramatically improves robustness to relearning, reducing regained accuracy by up to 50%, limiting it to 35% on Llama compared to 70%-76% for prior methods. Further analysis shows that the coherence cost is localized, affecting only a small set of concept-exclusive tokens. Our work establishes that precise embedding-level intervention is necessary for robust concept erasure, and demonstrates that existing methods can benefit from such augmentation.

### 1. 论文的核心问题与整体含义

- **研究背景**：大语言模型（LM）在实际部署中，需要能够安全、合规地擦除某些不适宜的知识（如危险、隐私或版权内容）。
- **现存问题**：主流的持久性知识擦除方法（如 RMU、CRISP）主要针对 MLP 层的参数进行更新。但越来越多的证据表明，这些方法无法彻底移除目标知识，残留知识可通过对抗性提示或重新学习（relearning）被恢复。
- **核心假设**：现有方法忽略了**嵌入层（embedding layer）**中编码的概念知识，这是导致擦除不彻底、鲁棒性差的关键瓶颈。
- **整体含义**：论文提出，要实现鲁棒的概念擦除，必须对 token 嵌入进行精准的干预，并为此设计了一种即插即用的增强模块 EMBER。

### 2. 论文提出的方法论 EMBER

- **核心思想**：EMBER 采用“先定位再编辑（localize-then-edit）”的策略，利用稀疏矩阵分解（Sparse MF）从 token 嵌入矩阵中剥离出与目标概念相关的特征，并精准地将其减去，而保留 token 的其他语义信息。
- **关键技术细节与流程**：
    1. **发现嵌入特征**：仅对目标概念句集及中性句集中出现的词汇对应的嵌入子矩阵E<sub>V*</sub>进行稀疏矩阵分解（E<sub>V*</sub><sup>T</sup> ≈ ZY）。通过对系数矩阵Y施加硬性“赢者通吃”（WTA）稀疏约束（保留每特征下系数最大的 1% token），迫使 Z 的每一列成为一个稀疏、可解释的“嵌入特征”，对应一小簇共享该方向的 token。
    2. **识别概念相关特征**：对每个特征 zi，计算其非零 token 在概念集与中性集上的系数绝对值之比 ρi（质量比）。只保留 ρi > τ 且经强语言模型判定与目标概念匹配的候选特征，构成概念特征集 F_C。
    3. **擦除概念特征**：对于每个概念专属 token t ∈ T_C，利用分解式 e_t = ZY_{:,t} + ε_t，将其重构时减去概念特征 F_C 的贡献：e_t ← e_t - δ Σ_{i∈F_C} Y_{i,t} z_i。通过超参数 δ 控制擦除强度（δ=1 时精确减去概念成分，δ>1 实现过度擦除以增强效果）。
- **与现有方法集成**：EMBER 作为即插即用模块，直接编辑输入嵌入矩阵，可与任何针对 MLP 层的方法（RMU、CRISP、PISCES）或论文提出的 SNMF（一种对 MLP 激活做半非负矩阵分解并定向消融权重的变体）结合使用。

### 3. 实验设计

- **使用模型**：Gemma-2-2B-it 和 Llama-3.1-8B-Instruct。
- **任务与概念**：在 18 个多样化概念上进行评估，涵盖虚构作品（如哈利·波特）、历史事件（如二战）、安全/敏感话题（如大麻、色情）等。其中 11 个来自先前工作，7 个为新增。
- **评估基准与指标**：
    - **效能（Efficacy）**：目标概念问答准确率（越低越好），包含**多项选择（MC）**和**开放生成（OE）**两种格式。
    - **特异性（Specificity）**：邻近非目标概念的相似领域准确率，以及 MMLU 基准分数。
    - **连贯性（Coherency）**：使用 AlpacaEval 评测指令遵循和流畅度。
    - **鲁棒性（Robustness）**：对擦除后模型用概念相关段落进行微调（重新学习），再测目标概念准确率（重新学习准确率）。
    - 汇总指标：使用调和平均分（HScore）综合评估效能、特异性和连贯性的权衡。
- **对比方法**：
    - 基础 MLP 擦除方法：RMU、CRISP、PISCES，以及论文提出的 MLP 变体 SNMF。
    - **与 EMBER 的集成**：上述各方法单独使用，以及各自与 EMBER 结合（+EMBER）。
    - **嵌入编辑基线**：Mean（用平均嵌入替换）和 Noise（按 EMBER 编辑强度注入同等大小的随机噪声），用以证明 EMBER 的收益来源于精准的方向编辑而非任意扰动。

- **实验规模**：在 2 个模型 × 18 个概念上，对 5 种基础擦除方法及其与 EMBER 的组合（以及 2 种嵌入编辑基线）进行了全面测试。每种配置均经过独立的超参数验证与最终测试集评估。

    - 实验从多个维度（MC 和 OE 准确率、相邻域准确率、MMLU、连贯性、重新学习鲁棒性）进行全面评估。
    - 超参数对所有方法均通过统一的验证集和 HScore 进行筛选，保证了比较的公平性。

### 6. 论文的主要结论与发现

- **嵌入层是关键瓶颈**：现有 MLP 中心的方法未能彻底移除知识，EMBER 通过编辑嵌入层显著改善了该问题。
- **效能与特异性提升**：在 MC 任务上，EMBER 平均降低概念准确率 8.1 个百分点，同时普遍提升相似领域准确率，对 MMLU 影响甚微，连贯性损失极小。
- **重新学习鲁棒性极大增强**：EMBER 使 RMU 和 CRISP 的重学习准确率在 Gemma 上降低 16%-24%，在 Llama 上降低 31%-33%。全 MF 管道（SNMF+EMBER）更是将 Llama 上的重学习准确率从 70%-76% 压低至 35%，降低约 50%。
- **编辑精准且局部**：EMBER 仅修改极少量 token 的嵌入（最多占词汇表的 0.14%），且连贯性下降主要集中在概念专有性的高 TF-IDF token 上，对通用 token 影响很小。
- **格式泛化性**：为 MC 优化的 EMBER 在开放生成（OE）评估中依然能带来相似领域的提升，表明其编辑的是概念知识而非格式捷径。

### 7. 优点

- **方法创新**：首次将知识擦除的焦点系统性地转移到嵌入层，并提供了一种基于稀疏矩阵分解的精准编辑方案。
- **即插即用的增强性**：EMBER 模块化设计，可无缝增强各类现有 MLP 擦除方法，无需修改原有方法流程。
- **鲁棒性突破**：在抗重新学习方面取得了显著超越以往方法的成果，直指知识擦除的浅层性问题。
- **评估全面扎实**：覆盖多模型、多概念、多任务格式，并增加了 Token 级局部性分析，研究深度高。
- **对比公正**：通过 Noise 和 Mean 基线，有力证明了方法增益来自特定的方向性编辑，而非任意扰动。

### 8. 不足与局限

- **语言单一性**：方法和评估均基于英语 Wikipedia 数据，编辑仅对英文 token 有效，无法抑制通过其他语言 token 访问同一概念。
- **仅编辑输入嵌入**：未编辑输出嵌入（unembedding），在理论上存在通过逆向复制未编辑的输出权重来绕过擦除的风险（适用于输入输出嵌入不共享的模型）。
- **概念依赖性**：该方法假设概念知识在嵌入中具有可发现的因子，对于罕见或极狭窄的概念，嵌入可能不是有效的擦除目标。
- **特征选择噪声**：基于语料共现的 token 标注可能引入噪声，导致部分语义无关 token 被编辑。
- **伦理双重用途**：强大的、难以逆转的擦除技术可能被恶意用于压制合法信息。
- **缺乏大规模/多语言验证**：实验限于 2B-8B 参数规模，在多语言、更大模型上的泛化性尚待验证。

## 摘要
对世界的表征，可以说，既包含特征信息（例如某物是蓝色的，某物是圆形的），也包含哪些特征属于同一对象的信息（例如这个圆形是蓝色的），我们称之为绑定信息。任何能够理解包含多个对象的场景的系统都必须解决绑定问题：它需要知道哪些特征属于一起。然而，尽管有研究表明视觉Transformer（ViTs）知道哪些图像块属于一起，但目前尚不清楚当前的深度学习模型是否学会了表征绑定信息，即特征层面的绑定。我们可能会认为绑定信息并不多，毕竟将特征错误归因到错误对象是ViT架构常见的问题，尤其是在对象共享特征的场景中。在此，我们通过信息论方法形式化绑定问题，并引入一种探测方法来测量模型表征中的绑定信息。我们在ViT上进行实验，测量架构不同组件的绑定信息，如图像摘要令牌[CLS]或空间令牌。我们使用具有不同绑定挑战的数据集，如特征共享、遮挡和自然特征，同时比较多个预训练ViT的性能。总体而言，我们的研究表明绑定是强视觉识别和推理的关键要素。

## Abstract
Representations of the world, arguably, contain information about features (e.g. something is blue, something is a circle) but also information about which features are part of the same object (e.g. the circle is blue), which we call binding information. Any system with the ability to understand scenes with multiple objects must be able to solve the binding problem: it needs to know which features belong together. However, despite work showing that Vision Transformers (ViTs) know which patches belong together, it is not known whether current deep learning models learn to exhibit binding information, i.e., for features. We may believe that there is not much binding information, after all misattributing features to wrong objects is a common failure of ViT-based architectures, especially in scenes with objects sharing features. Here we formalize the binding problem with an information-theoretic approach, and introduce a probing method to measure binding information in model representations. We perform experiments on ViTs, measuring binding from different components of the architecture, such as the image summary token [CLS] or the spatial tokens. We use datasets with different binding challenges, such as feature sharing, occlusion, and natural features, while comparing the performance of several pre-trained ViTs. Overall, our research demonstrates binding as a key ingredient to strong visual recognition and reasoning.

### **论文提出的方法论**

*   **核心思想：信息论形式化与线性探测**
    *   基于信息论，将绑定能力形式化定义为模型表征 `Z` 中所包含的关于场景中“哪些对象存在或不存在”（即**对象代码 `O`**）的信息量。
    *   通过训练**探针**从冻结的模型表征中解码出对象代码，并利用其交叉熵损失来估计条件熵，从而间接计算出绑定信息。

*   **关键技术细节与定义**
    *   **特征代码 (`F`)**: 一个二元随机向量，表示场景中哪些离散特征存在。
    *   **对象代码 (`O`)**: 一个二元随机向量，表示场景中哪些对象存在（对象被定义为一组特征的特定组合）。
    *   **绑定信息 (`BO(Z)`)** 定义为对象代码 `O` 与模型表征 `Z` 之间的互信息：`I(O; Z) = H(O) - H(O | Z)`。这可以理解为，知道表征 `Z` 后，对象代码 `O` 的不确定性减少了多少。
    *   **条件绑定信息 (`B* O, F(Z)`)**：为了剥离特征信息的影响，进一步定义了在已知特征代码 `F` 的条件下，表征 `Z` 关于对象代码 `O` 的互信息：`I(O; Z | F)`。
    *   **探针估计流程**:
        1.  **计算数据集先验**: 独立计算数据集的对象代码熵 `H(O)` 和特征代码熵 `H(F)`。
        2.  **训练探针**: 为了避免标签空间的指数爆炸，将对象代码 `O` 分解为自回归形式，为每个对象 `ok` 训练一个探针 `qθ(ok | o<k, z)`。同时，也训练特征探针 `qθ(fk | f<k, z)`。
        3.  **估计条件熵**: 使用探针在测试集上的交叉熵损失 `LCE(θ)` 作为对应条件熵 `H(O|Z)` 和 `H(F|Z)` 的**上限估计**（根据定理，`LCE(θ) = H(O|Z) + D(p||q) ≥ H(O|Z)`）。
        4.  **计算绑定信息**: 将估计出的条件熵和数据集先验代入公式，计算出 `BO(Z)` 和 `B* O, F(Z)`。

### **论文的主要结论与发现**

1.  **[CLS]令牌编码的绑定信息有限**: 图像摘要令牌 `[CLS]` 仅编码了**不到一半**的总绑定信息。尽管它对特征信息编码较好，但其对象绑定信息（条件绑定信息）仅占可解码总量的约 **42.4%**。
2.  **[CLS]令牌中绑定信息的结构**: `[CLS]` 中的绑定信息**主要是二次结构**，可以近似为颜色和形状特征的投影的点积。DNN探针相比二次探针提升不大，表明更高阶的交互作用贡献的信息极少。
3.  **空间令牌几乎完美编码绑定信息**: 全量的空间令牌集合包含了**近乎完美（约92.2%）** 的绑定信息。一种简化的注意力探针可以轻松解码这些信息，其注意力权重能够准确路由到目标对象所在的图像块。
4.  **数据复杂度影响绑定**: 随着特征空间（颜色和形状的数量）增大，模型表征中可解码的绑定信息比例下降，但并非指数级衰减。
5.  **遮挡阻碍绑定**: 场景中的遮挡程度越高，模型表征中可解码的绑定信息越少。
6.  **框架的普适性**: 该信息论框架能够有效应用于合成和自然数据集，并捕捉到不同模型（DINOv2, CLIP）和输入分辨率（224px vs. 336px）带来的绑定能力差异。

### **优点**

*   **形式化定义清晰**: 首次用严格的信息论语言（互信息、熵）形式化地定义了绑定问题，提供了可量化的测量标准，优于此前基于下游任务准确率的“症状式”评估。
*   **分析方法新颖**: 通过探测不同架构组件（`[CLS]` vs. 空间令牌）和探针结构（线性、二次等），深入分析了绑定信息在模型内部的存储方式和结构，尤其是发现 `[CLS]` 令牌中信息是二次的，这为模型设计提供了深刻洞见。
*   **测量框架实用**: 提出的基于探针的测量框架是通用的，原则上可以应用于任何模型、任何表征和任何预定义的特征/对象集合。

### **不足与局限**

*   **依赖离散特征词汇**: 核心方法要求对特征和对象进行预定义和离散化，这使得该框架难以直接应用于连续特征或未定义清晰概念集的场景。
*   **探针测量的固有局限**: 该方法测量的是“可解码”的绑定信息。高探针性能可能反映了模型表征中存在的潜在信息，但这**不一定意味着模型在下游任务推理时会因果性地使用这些信息**。
*   **条件定义的假设**: 条件绑定信息 `B* O,F(Z)` 的计算假设了对象代码 `O` 可以完全决定特征代码 `F`，这在某些嘈杂的生物学或人类感知场景中可能不成立。
*   **模型覆盖范围有限**: 实验仅在少数几个预训练的ViT模型上进行，未能涵盖其他主流架构（如CNN、ResNet）或最新的多模态大模型。

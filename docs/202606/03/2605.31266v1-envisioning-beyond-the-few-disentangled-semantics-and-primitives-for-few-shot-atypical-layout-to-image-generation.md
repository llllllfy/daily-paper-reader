## 摘要
布局到图像（L2I）任务通过对象类别与空间布局实现对图像生成的细粒度控制。然而，现有的L2I方法在小样本非典型场景下会产生破碎和扭曲的生成结果。我们将这种失败称为表示碎片化，源于语义身份与视觉细节之间的粒度不匹配，导致彼此纠缠。为解决这一问题，我们提出了一种表示驱动的框架，将语义与基元解耦，以实现稳健的小样本适应。具体而言，语义锚定将类别语义聚合为锚点以稳定身份，而基元注入则建模可重组基元以进行稳健的局部细节建模。概念引导进一步通过显著性感知目标来调节优化，以保持前景语义一致性。大量实验表明，在多样化的非典型领域中，5样本设置下的视觉保真度和对齐度均优于现有的最先进L2I方法。源代码已公开发布于https://github.com/iCVTEAM/DSP。

## Abstract
The layout-to-image (L2I) task enables fine-grained control over image generation via object categories and spatial layouts. However, existing L2I methods yield fragmented and distorted generations under few-shot atypical settings. We term this failure as representation fragmentation, arising from a granularity mismatch that entangles semantic identity with visual details. To address this issue, we propose a representation-driven framework that disentangles semantics from primitives for robust few-shot adaptation. Specifically, Semantic Anchoring aggregates categorical semantics into anchors for stable identity, while Primitive Imbuing models recomposable primitives for robust local detail modeling. Conceptual Steering further regulates optimization with a saliency-aware objective to preserve foreground semantic consistency. Extensive experiments demonstrate consistent improvements in the 5-shot regime over state-of-the-art L2I methods in both visual fidelity and alignment across diverse atypical domains. The source code is publicly available at https://github.com/iCVTEAM/DSP.

## 1. 论文核心问题与研究动机

*   **核心问题**：论文关注在**小样本非典型视觉场景下的布局到图像（L2I）生成任务**。
*   **现象与定义**：现有 L2I 方法在只有少量样本（如 5 张）可用的非典型领域（如水下、低光、航拍图像）中，常生成**纹理破碎、几何变形**的图像。作者将此失败模式定义为**表示碎片化（Representation Fragmentation）**。
*   **问题根源**：研究认为碎片化的根本原因是**语义粒度不匹配（Granularity Mismatch）**。高层语义身份需要全局一致性和跨实例稳定性，而低层视觉纹理与外观细节则是局部的、高度可变的。现有方法未能有效分离这两种异构粒度，导致**全局语义被局部细节噪音所“纠缠”和“淹没”**。
*   **研究目标**：提出一个**表示驱动的框架**，通过**显式解耦语义级表示和局部基元级表示**，来解决上述碎片化问题，实现稳健的小样本 L2I 生成。

## 2. 方法论：语义-基元解耦框架

论文提出了一套由三个紧密耦合组件构成的表示驱动框架，其核心思想是**解耦语义身份与视觉细节**。

### 2.1 语义锚定
*   **目的**：在小样本条件下稳定类别语义身份，防止语义漂移。
*   **机制**：
    1.  构建**样本池**，裁剪出所有目标物体的样本。
    2.  使用冻结的 DINOv2 编码器提取类别级密集特征，并通过 Resampler 和自注意力机制聚合为**语义锚点**，捕获了跨样本的语义共识。
    3.  通过一个**门控交叉注意力机制**，将锚点信息注入到 U-Net 的前景嵌入中，从而在生成时诱导稳定、一致的类别语义。

### 2.2 基元注入
*   **目的**：在语义锚定基础上，对小样本下难以估计的细粒度局部视觉细节（如纹理、几何片段）进行稳健建模和生成。
*   **机制**：
    1.  定义**视觉基元**为一组可学习的原型嵌入 `P`。
    2.  通过**交替最小化**来学习基元：固定 `P`，用**岭回归（Ridge Regression）**求系数矩阵 `W` 来重构目标特征 `T`；再固定 `W`，优化 `P` 以最小化重构误差 `||T - WP||^2_F`。岭回归的闭合解形式保证了小样本下的数值稳定性。
    3.  将学习到的基元 `P` 通过一个**空间门控交叉注意力机制**注入到 U-Net 的深层解码层，以精细化局部细节。

### 2.3 概念引导
*   **目的**：在优化过程中，显式强制模型关注前景区域，防止模型走捷径去拟合背景模式而忽略新颖物体的语义保真度。
*   **机制**：
    1.  采用**文本驱动的 GradCAM** 分别计算生成图像与真实图像关于目标类别的激活图。
    2.  基于激活图差异，构建一个**空间惩罚掩码** `Ω`，对语义不一致区域施加更大的损失权重。
    3.  最终的扩散模型训练损失变为：`L_final = E[Ω ⊙ ||ε_Θ - ε||^2_2]`。

### 2.4 整体流程
1.  **基础训练阶段**：在大量源域数据上预训练一个标准的潜在扩散模型，获得通用布局生成能力。
2.  **小样本自适应阶段**：冻结基础模型所有权重，仅利用少量目标域样本（如 5 张）来优化新加入的语义锚定、基元注入和概念引导模块的参数。

1.  **有效性**：在小样本非典型 L2I 生成中，**显式解耦语义与视觉基元**是缓解表示碎片化的有效途径。

## 摘要
弱监督增量语义分割 (WILSS) 面临持续引入噪声监督的问题，这会逐步腐蚀类别级表征，导致严重的特征漂移和语义污染，从而使得新学习的类别覆盖旧类别。为解决这些问题，我们提出了一种名为 SASA 的抗漂移 WILSS 方法，旨在通过语义锚点与空间仲裁来稳定语义学习。具体来说，在表征层面，我们引入可学习令牌的语义锚点作为刚性的类别级引用，以保持长期的语义身份。作为补充，弹性残差自适应机制实现了有控制的实例特定细化，确保了稳定而灵活的学习轨迹。在监督层面，我们开发了一种空间标签仲裁机制，通过几何感知决策直接过滤不可靠信号，并强制执行“一个对象，一个类别”的严格约束。通过协同稳定表征并提高监督可靠性，SASA 有效减轻了弱监督下的特征漂移。在标准基准上的大量实验表明，我们的方法始终优于现有的最先进方法，尤其是在具有挑战性的多步增量设置中。代码可在 https://github.com/ZhonggaiWang/SASA 获取。

## Abstract
Weakly Incremental Learning for Semantic Segmentation (WILSS) suffers from the continuous introduction of noisy supervision, which progressively corrupts class-level representations, leading to severe feature drift and semantic corruption, thereby causing newly learned classes to overwrite old ones. To address these issues, we propose a drift-resilient WILSS approach, named SASA, designed to stabilize semantic learning via Semantic Anchors and Spatial Arbitration. Specifically, at the representation level, we introduce semantic anchors of learnable tokens as rigid class-level references to preserve long-term semantic identity. Complementary to this, an elastic residual adaptation facilitates controlled, instance-specific refinement, ensuring a stable yet flexible learning trajectory. At the supervision level, we develop a Spatial Label Arbitration mechanism that performs geometry-aware decisions to directly filter unreliable signals and enforce a strict "one object, one class" constraint. By synergistically stabilizing representations and improving supervision reliability, SASA effectively mitigates feature drift under weak supervision. Extensive experiments on standard benchmarks demonstrate that our approach consistently outperforms existing state-of-the-art methods, particularly in challenging multi-step incremental settings. The code is available at https://github.com/ZhonggaiWang/SASA.

### 1. 论文核心问题与研究动机
弱监督增量语义分割（WILSS）在真实场景中极具实用价值，因为每步增量只需图像级标签。然而，该设定面临一个根本性难题：**持续的噪声监督会逐渐腐蚀类别级表征**，导致灾难性遗忘和类间覆盖。具体表现为：
- 新类的弱定位信号（如CAM）通常只激活对象的局部判别区域，大量区域仍需由冻结的旧模型填补，但旧模型无法分辨新类，往往会将其误标为视觉相似的已知类，从而在同一对象实例内引入矛盾的监督信号。
- 这种矛盾噪声持续注入，迫使新类特征向旧类分布偏移，同时扰动旧类的已有表征，最终造成**特征漂移**，决策边界严重退化。
现有方法或仅改善伪标签质量，或只依赖蒸馏/正则化保护旧知识，都无法从根本上阻止语义参照系本身的失稳。因此，作者提出需要一种既能**刚性锚定类别语义**又能**弹性适应实例差异**，并在**监督层面直接滤除空间结构噪声**的新框架。

### 2. 方法论：SASA 框架的核心思想与技术细节
SASA 通过**表示层**和**监督层**的双重设计，协同抵抗特征漂移。

#### 2.1 表示层：抗漂移语义锚点（DSA）
- **刚性语义锚点（Rigid Anchors）**：为每个类别维护一个可学习向量 \(A_t^k \in \mathbb{R}^D\)，作为该类别在嵌入空间中的长期固定参照，不受当前噪声伪标签直接冲击。这些锚点通过优化保证类间可分性，形成稳定的全局语义坐标。
- **弹性残差令牌（Elastic Residual Tokens, ERT）**：为了容纳实例级外观变化，动态生成残差 \(R_t\)。通过交叉注意力机制，以锚点 \(A_t\) 为查询、编码器输出的扁平化特征图 \(\hat{F}\) 为键和值，计算得到残差令牌。最终表示 \(Z_t = A_t + R_t\)，仅通过“小步”弹性调整，避免噪声直接写入类语义。同时用 \(\ell_2\) 正则 \(L_{reg}\) 限制残差幅度。
- **分离约束与知识蒸馏**：对最终令牌施加分离损失 \(L_{sep}\)，鼓励其与自身锚点对齐、与其他锚点正交；对旧类锚点施加蒸馏损失 \(L_{dist}\)，使其与前一阶段的锚点保持一致，防止灾难性遗忘。密集预测使用像素特征与最终令牌的余弦相似度产生分数，由交叉熵损失 \(L_{seg}\) 监督。

#### 2.2 监督层：空间标签仲裁（SLA）
目标是在训练时用几何先验**过滤不可靠的伪标签**，强制执行“一个对象，一个类别”原则。
- **空间可靠性图**：利用类无关分割器（如SAM）生成对象掩码 \(M\)。对每个掩码，以质心为中心，通过高斯衰减函数计算像素可靠性权重 \(w_p\)，侧重于对象语义核心区域，抑制边界噪声。
- **加权仲裁与标签融合**：先计算每个掩码内新类信号的激活密度 \(\rho_i\)（按权重求和）。然后进行加权多数投票：若 \(\rho_i > \tau\)，候选类集为新类 \(C_t\)；否则为旧类 \(C_{0:t-1}\)。投票胜出的类别标签 \(\hat{y}_i\) 将一致赋予该掩码内所有像素，生成精炼后的伪标签，用于驱动分割训练。

最终损失函数为：
\[
L = L_{cls} + \lambda_{seg} L_{seg} + \lambda_{sep} L_{sep} + \lambda_{dist} L_{dist} + \lambda_{reg} L_{reg}
\]
其中 \(L_{cls}\) 为用于生成CAM的图像级分类损失。
